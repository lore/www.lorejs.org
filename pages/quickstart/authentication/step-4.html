Import the `src/utils/auth` utility and update the `onAuthentication` callback to look like this: {% tabs tab1={"name": "ES5", "id": 101} | tab2={"name": "ES6", "id": 102} | tab3={"name": "ESNext", "id": 103} %} {% tab id=101, is_active=true %} ```jsx var auth = require('../utils/auth'); ... onAuthentication: function(authResult) { auth.saveToken(authResult.idToken); }, ... ``` {% endtab %} {% tab id=102 %} ```jsx import auth from '../utils/auth'; ... onAuthentication(authResult) { auth.saveToken(authResult.idToken); } ... ``` {% endtab %} {% tab id=103 %} ```jsx import auth from '../utils/auth'; ... onAuthentication(authResult) { auth.saveToken(authResult.idToken); } ... ``` {% endtab %} {% endtabs %} With this change in place, whenever the user logs in we will save the idToken provided by Auth0 to localStorage. ### Redirect the User With the token saved to localStorage, we can now redirect the user back to the application, and (because the token now exists) it won't redirect us back. Update the `onAuthentication` method to look like this (router is provided as a prop by React Router): {% tabs tab1={"name": "ES5", "id": 104} | tab2={"name": "ES6", "id": 105} | tab3={"name": "ESNext", "id": 106} %} {% tab id=104, is_active=true %} ```jsx onAuthentication: function(authResult) { auth.saveToken(authResult.idToken); this.props.router.push('/'); }, ``` {% endtab %} {% tab id=105 %} ```jsx onAuthentication(authResult) { auth.saveToken(authResult.idToken); this.props.router.push('/'); } ``` {% endtab %} {% tab id=106 %} ```jsx onAuthentication(authResult) { auth.saveToken(authResult.idToken); this.props.router.push('/'); } ``` {% endtab %} {% endtabs %} Finally, add a `componentWillUnmount` method to hide the lock when the `Login` component is unmounted: {% tabs tab1={"name": "ES5", "id": 107} | tab2={"name": "ES6", "id": 108} | tab3={"name": "ESNext", "id": 109} %} {% tab id=107, is_active=true %} ```jsx componentWillUnmount: function() { this.lock.hide(); }, ``` {% endtab %} {% tab id=108 %} ```jsx componentWillUnmount() { this.lock.hide(); } ``` {% endtab %} {% tab id=109 %} ```jsx componentWillUnmount() { this.lock.hide(); } ``` {% endtab %} {% endtabs %} With that change in place, refresh the browser and navigate to `/login`. Once you login in, the application will redirect you to the `/` route and transition the login dialog off the page.
