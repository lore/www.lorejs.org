### What You'll Build This is the application we'll be building. It is a Twitter-like app, pre-popuated with characters and quotes from [Chrono Trigger](https://en.wikipedia.org/wiki/Chrono_Trigger).
### Concepts Covered This quickstart will cover the following concepts: * Creating and building a React application * Layout out your application visuals with mock data * Routing * Fetching data from an API server * Populating your application with real data * Launching dialogs * Authentication (logging users in) * Authorization (restricting what users see) * Resolving breaking API changes * Pagination * Infinite Scrolling * Filtering ### Questions/Issues If you have any questions or concerns while navigating through the quickstart, please [submit an issue on GitHub](https://github.com/lore/lore/issues).
